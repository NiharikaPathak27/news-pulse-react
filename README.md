#  NewsPulse-React – News Web Application

A modern and responsive news web application built using **React.js**.
It fetches real-time news using the **News API** and displays them in an interactive and dynamic UI.

---

##  Features

* 🔴 Live news fetching using News API
* 🔍 Search functionality for custom queries
* 📂 Category-based filtering (Cricket, India, Technology, Politics)
* ♾️ Infinite scroll (coming soon)
* 🌙 Dark mode toggle
* ⚡ Fast and dynamic UI using React components
* 📱 Fully responsive design

---

##  Tech Stack

* React.js
* HTML5
* CSS3
* JavaScript (ES6+)
* News API

---

##  How to Run

1. Clone the repository
2. Navigate to the project folder
3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```
